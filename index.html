Now here comes the real challenge
A list is first created with name s_arr
A for loop runs from 0 < len(flag)-1; so if len of flag = 38; then the loop runs from 0 to 37

in this line 

s_i = sum([arr[j]*ord(FLAG[j]) for j in range(l)])
we loop through each character in the flag and then multiple its ascii value by the corresponding value in arr with the same index
eg if flag="hello", and randomNumbers(in arr) = [2,4,7], then we get 
ord(h)*2 + ord(e)*4 + ord(l)*7
After that we sum them and append their result to s_arr
Then the random values in arr is shifted one step forward eg [2, 5, 7] to [7, 2, 5] for the next iteration


So now in order to solve this challenge I decided to print down all the shifting of the the values in arr during the loop process,
and i got this, usiing the code below 

      
        for i in range(len(FLAG) - 1):
          print(arr)
          arr = [arr[-1]] + arr[:-1]

Yes we can now see a matrix (system of equations) forming with the characters of our flag being the variables

The final result is a 37 x 38 matrix since the loop run for 0 to len(flag - 1) = 37

and thier corresponding sum 1 x 37

Even though we can use numpy to solve this, since the matrix formed is not a square matrix it would be impossible

to find all the characters of our flag, in this case the variables.

This is where I realised the topic for the challege "Just one more"

WE only need one more equation to form 38 x 38 matrix in order to solve this challege but the loop eneded 1 step before


So my solution was, first i know the flag starts with letter f, this means if I take the ascii value of f and multiple it by its corresponding constant in all the equation and then subtract the result from the sum, i should be able to come out with somethiing to solve this challenge,

Since i would be subtracting ord(f) * rnd from the sum, then i have to remove it from the equation 


      
ord(l)*rnd2 + ord(a)*rnd3 + ord(g)*rnd4 ... = s_arr[0] - ord(f)*rnd1 
ord(l)*rnd1 + ord(a)*rnd2 + ord(g)*rnd3 ... = s_arr[0] - ord(f)*rnd4
ord(l)*rnd4 + ord(a)*rnd1 + ord(g)*rnd2 ... = s_arr[0] - ord(f)*rnd3
ord(l)*rnd1 + ord(a)*rnd4 + ord(g)*rnd3 ... = s_arr[0] - ord(f)*rnd2  continued.....
